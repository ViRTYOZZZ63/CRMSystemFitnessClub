# PulsePoint CRM — обновлённая концепция

## Ключевые улучшения
- Все данные переведены на хранение в БД SQLite (`crm.db`) через backend API (`/api/bootstrap`, `/api/login`, `/api/register`, `/api/state`).
- Авторизационные данные (email/пароль/роль) также хранятся в базе данных, а не в localStorage.
- Дополнительно введена нормализация email и проверка дублей (регистрация и админ-создание учеток) для устойчивой синхронизации данных после merge.
- Синхронизирована структура модулей и стилей после разрешения конфликтов ветки (auth, dashboard, mobile-nav).
- Полностью переработана страница авторизации: вход только по email/паролю, роль подтягивается автоматически из учётной записи.
- Добавлен сценарий создания новых учётных записей прямо на auth-странице.
- Добавлено отдельное админ-управление учётками внутри CRM.
- Переработано мобильное меню: адаптивный burger c открытием вкладок ролей.
- Визуал усилен: dark premium, glassmorphism, плотная информационная архитектура.

## Ролевые кабинеты

- KPI блока клуба,
- диаграмма загрузки тренеров,
- календарь нагрузки на каждый день месяца,
- управление тренерами,
- полноценное расписание занятий,
- управление учётками,
- зарплатная ведомость.

### Тренер
- персональный дашборд,
- управление статусом занятий,
- клиенты и заметки,
- рабочее время.

### HR
- обзор команды,
- воронка найма,
- сертификации,
- посещаемость и отработанные часы.

- прогноз дохода.

- Расширенный набор тренеров, расписания, платежей и кандидатов.
- Привязка части данных между ролями: расписание ↔ нагрузка ↔ выплаты ↔ рабочее время.
